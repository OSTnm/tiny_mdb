# Tiny Mdb
> **Tiny Mdb**, simple tool for mdb data processing.

Tiny Mdb generate csv based on mdb with builtin policies.

| Platform                       | Test Result |
| ------------------------------ | ----------- |
| Win10 VM                       | ✔           |
| Mac                            | ✔           |

## Usage

- put mdb files to mdbs/
- python tiny_mdb.py
- csv will be generated in outputs
